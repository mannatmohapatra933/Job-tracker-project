🚀 Job Application Tracker

A full-stack Job Application Tracker web app to manage job applications, wishlist, notes, and analytics.

---

🔗 Live Demo

🌐 Frontend: https://job-tracker-project-beryl.vercel.app/
⚙️ Backend API: https://job-tracker-project-3b2m.onrender.com/api/jobs

---

✨ Features

- 🔐 User Authentication (JWT based login/signup)
- 📌 Add, edit, delete job applications
- ⭐ Wishlist feature
- 📝 Add notes for each job
- 📊 Analytics dashboard
- 🔍 Filter by company, experience, and location
- 🌐 Multi-user support (user-specific data)

---

🛠 Tech Stack

Frontend:

- React.js
- CSS

Backend:

- Spring Boot
- JWT Authentication
- REST APIs

Database:

- MySQL / PostgreSQL

Deployment:

- Frontend → Vercel
- Backend → Render

---

⚙️ Environment Variables

Frontend (.env)

REACT_APP_API_URL=https://job-tracker-project-3b2m.onrender.com

---

🚀 How to Run Locally

Frontend:

cd react-app
npm install
npm start

Backend:

cd java-backend
./mvnw spring-boot:run
